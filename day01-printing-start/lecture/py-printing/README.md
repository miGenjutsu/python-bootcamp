## Learning Path

- printing multiple lines
    ```sh
    print("Hello World!")
    print("Hello World")
    print("Hello World")
    ```
- same method but saving extra typing
  ```sh
  print("Hello World!\nHello World!\nHello World!")
  ```
- concatenate strings
  ```sh
  print("Hello" + " Developer")
  ```
